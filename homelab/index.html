homelab

A Docker-based home server running media automation, game servers, smart home, networking, and monitoring services. This website is served from it.

architecture

Everything runs on an Intel i9-13900H box with 96 GB of RAM, an NVMe boot drive, a secondary NVMe for backups and miscellaneous hot storage, and an 80 TB RAID5 array across four drives in a DAS enclosure. About 50 containers are split across 5 Docker Compose stacks and 8 subnets. The main stack handles media, monitoring, and home automation, while Authentik, Headscale, Nextcloud, and Nginx each get their own isolated stack.

Nginx sits at the edge, connected to every application network, terminating TLS with wildcard Let’s Encrypt certificates for two domains via Cloudflare DNS challenges. Internal services are locked behind Tailscale (coordinated by a self-hosted Headscale instance) while Plex, Overseerr, the Authentik outpost, and the info and game server pages are the only public-facing endpoints. All torrent and indxer traffic routes through a Gluetun container running a WireGuard tunnel to ProtonVPN.

automation

The stack is orchestrated by a systemd service that boots containers in 4 phases: core infrastructure first, then isolated network stacks, followed by nginx once all networks exist, and finally a stabilization phase that pauses media syncs until the VPN is healthy and indexers are reset.

A fleet of cron jobs handles the fragile parts: monitoring VPN health every few minutes and restarting dependent containers when it drops, clearing stalled downloads, resetting indexer backoff after rate limiting cascades, syncing media requests, falling back from 4K to 1080p when releases aren't available, searching for missing content periodically, and keeping Sonarr and Radarr quality profiles and custom formats in sync with community guides via Recyclarr.

Container updates happen via Watchtower nightly, except for VPN-dependent containers which update through a dedicated script that handles the restart choreography without killing the network for everything else. Autoheal runs alongside it, automatically restarting any container that fails its health check. Duplicati backs up configs and key data to the secondary NVMe on a nightly schedule.
