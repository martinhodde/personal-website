projects

Projects in computer graphics and computer vision. Each links to a full writeup with results.

Path Tracer
Physically-based path tracer with BVH acceleration, direct and global illumination via Monte Carlo integration, Russian Roulette termination, and support for mirror, glass, and microfacet materials with importance-sampled BRDFs.
C++ rendering global illumination
Neural Style Transfer
Renders the content of one image in the artistic style of another using a modified VGG-19 network. Fuses content and style representations by minimizing combined Gram matrix style loss and feature activation content loss.
Python deep learning optimization
Cloth Simulator
Real-time cloth simulation using mass-spring systems with Verlet integration. Sphere, plane, and self-collision handling with spatial hashing. GLSL shaders including Blinn-Phong, texture/bump/displacement mapping, and environment reflections.
C++ simulation shaders
Auto-Stitching Photo Mosaics
Panorama stitching from manual and automatic correspondence. Homography estimation, Laplacian blending, Harris corner detection, ANMS feature selection, descriptor matching, and RANSAC for robust autostitching.
Python computer vision panorama stitching
Mesh Editor
Bezier curve/surface evaluation via de Casteljau, triangle mesh manipulation with the half-edge data structure. Implemented edge flip, edge split, and Loop subdivision for progressive mesh upsampling.
C++ geometry subdivision
Colorizing the Prokudin-Gorskii Collection
Reconstructed color images from 1900s glass plate negatives by aligning RGB channels using normalized cross-correlation with image pyramids. Includes white balancing, contrast enhancement, and edge-based alignment.
Python image processing image alignment
Face Morphing
Animated face morphs using Delaunay triangulation and affine warping. Computed average faces from the IMM Danish Face Dataset, then generated caricatures by extrapolating differences from the population mean.
Python computer vision image warping
Facial Keypoint Detection
Predicts facial landmarks using progressively sophisticated CNN architectures: simple CNNs on IMM data, ResNet-34 transfer learning on iBug (6666 images), and U-Net pixelwise heatmap classification.
Python deep learning transfer learning
Lightfield Camera
Simulated depth refocusing and aperture adjustment using Stanford Light Field Archive data. Shifting and averaging multi-view images enables post-capture focus control. Includes interactive click-to-refocus.
Python computational photography depth of field
Rasterizer
Software rasterizer with supersampled antialiasing (up to 16x), barycentric coordinate interpolation, texture mapping with bilinear filtering, and mipmap level sampling for efficient multi-resolution rendering.
C++ rendering antialiasing
Fun with Filters and Frequencies
Image processing through frequency analysis. Edge detection, unsharp masking, hybrid images (different at close vs. far viewing distances), and multiresolution blending via Gaussian/Laplacian stacks.
Python image processing frequency domain
