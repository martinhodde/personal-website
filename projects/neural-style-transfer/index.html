Neural Style Transfer

Overview

This project implements A Neural Algorithm of Artistic Style. The objective is to render the content of one image in the artistic style of another image. We adjust a pretrained VGG-19 convolutional neural network to serve as a collection of feature representations for a given input image. The higher convolutional layers of the network correspond to the high-level object contents of the image, while the upstream layers represent low-level texture information. This structural separation of image details in the network allows us to selectively fuse together distinct content and style aspects across images.

Model Architecture

Using the VGG-19 model as a baseline, we add an image normalization layer to the front, pluck out the convolutional layers, replace all of the max pooling layers with average pooling layers, insert content loss and style loss modules where appropriate, and exclude all of the linear layers. To obtain feature representations, we simply extract the convolutional filter activations.

Optimization

Let $\vec{p}$ and $\vec{x}$ be the original image and the generated image, respectively, and let $P^{\ell}$ and $F^{\ell}$ be their feature representations for layer $\ell$. The content loss is defined as the mean squared error between the content image and the appropriate feature activations. To compute the style loss, we use the Gram matrix $G^{\ell}$, where $G_{ij}^{\ell}$ is the inner product between the vectorized feature map $i$ and $j$ in layer $\ell$, and minimize the mean squared distance between the Gram matrices of the original and generated images.

To generate output images, we initialize the output image to the content image and simultaneously minimize the content and style loss for all relevant activation layers, using separate empirically determined weights $\alpha$ and $\beta$ for each type of loss. We run the LBFGS optimizer for 250 iterations with an $\alpha / \beta$ ratio of $5 \times 10^{-4}$.
